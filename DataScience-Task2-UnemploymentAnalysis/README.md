#  Unemployment Analysis with Python

##  Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on unemployment data in India to identify regional and temporal trends, with a special focus on the impact of the COVID-19 pandemic.

---

##  Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

##  Dataset

**Dataset:** Unemployment in India

The dataset contains information such as:

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

---

##  Project Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Region-wise Unemployment Analysis
- Monthly Trend Analysis
- State-wise Time Series Analysis
- Top 10 States by Average Unemployment Rate
- Correlation Analysis
- Pre-COVID vs Post-COVID Comparison

---

##  Visualizations

The project includes:

- 📌 Region-wise Average Unemployment Rate
- 📌 Monthly Unemployment Trend
- 📌 Time Series Analysis for Major States
- 📌 Top 10 States with Highest Average Unemployment
- 📌 Correlation Heatmap
- 📌 COVID-19 Impact Comparison

##  How to Run

1. Clone the repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open the notebook:

```bash
jupyter notebook Unemployment_Analysis.ipynb
```

---

##  Key Insights

- Unemployment rates varied significantly across different states.
- COVID-19 led to a noticeable increase in unemployment during 2020.
- Labour Participation Rate and Employment showed meaningful relationships with unemployment trends.
- Visual analysis helps identify the most affected regions and periods.

---

##  Conclusion

This project demonstrates how Python-based data analysis and visualization techniques can be used to understand unemployment trends in India and evaluate the impact of major events like the COVID-19 pandemic.
